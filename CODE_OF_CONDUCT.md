# Code of Conduct

## Our Pledge

We are committed to making participation in Navo welcoming, respectful, and constructive for everyone.

## Expected Behavior

Please:

- Be respectful.
- Assume good intent.
- Provide constructive feedback.
- Welcome different perspectives.
- Focus discussions on ideas rather than individuals.
- Help create a positive learning environment.

## Unacceptable Behavior

Examples include:

- Harassment or discrimination
- Personal attacks
- Trolling or inflammatory comments
- Spam or intentionally disruptive behavior
- Sharing private information without permission

## Enforcement

Project maintainers are responsible for interpreting and enforcing this Code of Conduct.

They may edit, remove, or reject contributions that violate these guidelines.

## Scope

This Code of Conduct applies to all project spaces, including GitHub Issues, Pull Requests, Discussions, and other official communication channels.
