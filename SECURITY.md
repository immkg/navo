# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do not create a public GitHub Issue.

Instead, report it privately by email.

**Email**

[mayankGupta690@gmail.com](mailto:mayankGupta690@gmail.com)

Please include:

- A description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation, if available

We will acknowledge reports as soon as possible and work toward a responsible resolution.

## Supported Versions

As Navo is currently under active development, only the latest version is considered supported for security fixes.
