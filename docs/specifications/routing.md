# Routing Specification

## Purpose

Routing determines the most effective sequence of work.

Unlike scheduling, routing optimizes execution.

## Optimization Factors

Routing may optimize for:

* distance
* time
* dependencies
* cost
* effort
* priority
* opportunities

## Rules

Routes are:

* dynamic
* context-aware
* explainable
* continuously recomputed

Multiple valid routes may exist for the same intention.

The user always chooses the final route.
