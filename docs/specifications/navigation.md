# Navigation Specification

## Purpose

Navigation answers:

> Given where I am now, what should I do next?

## Inputs

* Current location
* Current time
* Current context
* Available work
* Existing route

## Outputs

Navigation recommends:

* next action
* nearby opportunities
* work to postpone
* better execution paths

## Rules

Navigation considers:

* physical movement
* logical movement
* available time
* context changes
* current commitments

Navigation continuously updates as context changes.
