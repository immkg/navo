# Planner Specification

## Purpose

The Planner determines the best path toward an intention.

Its responsibility is to recommend, not control.

## Inputs

* Intentions
* Work Items
* Context
* Schedule
* Dependencies
* Preferences

## Outputs

The Planner produces:

* suggested work
* execution order
* grouped work
* deferred work
* planning recommendations

## Rules

The Planner:

* continuously adapts
* explains recommendations
* respects user decisions
* never modifies work automatically

## Goal

Reduce decision fatigue while maximizing meaningful progress.
