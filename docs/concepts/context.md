# Context

Context is everything that influences a decision.

The same work may be appropriate in one situation and unnecessary in another.

Examples of context include:

- Time
- Location
- Weather
- Energy
- Budget
- Available resources
- Nearby places
- People
- Deadlines
- Existing commitments

Context does not change the work itself.

It changes what makes sense to do next.

Navo continuously uses context to improve planning and navigation.
