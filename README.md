# Navo

> Navigate intentions, not just tasks.

## About

Navo is an intent-first planning system built around how people naturally think and work.

People rarely start with a checklist. They start with an intention such as planning a trip, buying groceries, preparing for an interview, or organizing an event. The details emerge over time through decisions, collaboration, changing priorities, and new opportunities.

Navo models this natural process. Instead of reducing everything to isolated tasks, it understands intentions, work, dependencies, people, locations, and time as parts of the same journey.

The goal is simple: software should adapt to the way people think, not the other way around.

## Core Concepts

### Intent

Every piece of work begins with an intention. An intent represents a desired outcome, not a predefined list of tasks.

### Work

Work is the building block of execution. It can represent a task, decision, reminder, approval, research, or any other action that moves an intent forward.

### Context

Work exists within context, including people, locations, time, resources, and constraints. Context influences what should happen next.

### Flow

Work is connected. It evolves through dependencies, collaboration, branching paths, and discovery rather than following a single linear checklist.

## What Navo Is

* An intent-first planning system.
* A graph of connected work instead of isolated tasks.
* A system that understands context, collaboration, and dependencies.
* A foundation for planning daily life, projects, travel, learning, and everything in between.

## What Navo Isn't

* A traditional to-do list.
* A flat checklist.
* A project management tool with rigid workflows.
* A calendar replacement.
